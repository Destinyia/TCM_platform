# 寸关尺三通道脉波纵向表征分析模块开发计划文档（更新版）

更新时间：2026-05-23
更新口径：前四阶段已完成；后续阶段以“特征提取后的患者级纵向分析、质量统计、设备适配、患者间差异和多模态关联”为主线。

## 1. 项目定位

本模块不再以“单次脉诊信号是否稳定”作为核心目标，而是以“是否采集到具有真实脉搏周期意义的寸、关、尺三通道脉波模式”为第一判断标准。

在实际采集中，同一用户的玉生堂脉诊数据可能出现明显差异：有时寸、关、尺三通道波形清晰，周期性强；有时信噪比很低，甚至某些通道虽然非常稳定，但波形幅度很小，可能是未对准寸关尺位置、接触压力不足、传感器佩戴角度偏移或固定尺寸传感器与患者手腕形态不匹配导致。因此，模块不能简单地把“稳定”视为“高质量”。最稳定的信号有可能只是没有采到有效脉搏信号。

本模块的核心目标是：以同一患者为纵向分析主体，将寸、关、尺三通道脉波分解为低频趋势、周期性脉搏模板、横向相位偏移和非周期残差，重点分析每个通道的周期化信噪比、去趋势波动性、模板一致性、压力/接触干扰、通道能量分布、相位变异、个人基线偏离度和设备适配风险，从而展示同一次记录内的波形模式稳定性、早中晚三次的状态差异，以及同一患者长期脉波状态的变化趋势。

本模块暂不直接输出“证候诊断”或“疾病判断”，而是输出可解释的三通道脉波状态表征。中医理论主要用于解释寸、关、尺三部脉波变化的观察维度，而不应直接替代医师诊断。

## 2. 当前实现基线

本节是当前开发和验收的权威实现口径。后续算法、接口、页面或阈值发生变化时，应优先更新本节，再更新阶段规划。

### 2.1 模块边界与数据平台关系

脉诊分析模块是数据平台的低耦合下游研究模块，不是平台通用模型能力的前置依赖。

数据平台负责从来源存储解析和固化标准化数据、文件资产、数据集版本和下载清单，提供面向脉诊模块以及其他独立研究者的通用数据集导出与下载能力，并保留脱敏、数据版本、来源证据、解析版本、审计和资产路径等平台职责。

脉诊分析模块负责消费平台导出的脉诊研究数据集或标准记录接口，独立执行周期信号、模板、残差、局部未对准、纵向趋势等分析，并将经过验证的算法产物复制并版本化集成到平台后端，用于在线展示及入库后自动分析。

### 2.3 当前在线质量判定口径

质量评价必须同时考虑周期信号、模板稳定性、信号质量和稳定片段比例。稳定片段比例或模板稳定分数单独较高，不能证明记录质量合格。

当某通道表现为波形稳定，但 `periodic_snr` 低、`pulse_energy` 低或疑似未对准评分高时，页面必须提示该通道可能未对准或接触不足，不应判为高质量稳定模板。

当前页面用于在线展示和通道组合统计的阈值如下：

| 维度 | 通过规则 | 说明 |
| --- | --- | --- |
| 周期 SNR | `periodic_snr > 0.4055`，页面显示为 `> 0.406` | 基于 2026-05-23 当前总体记录分布校准 |
| 模板稳定性 | `template_stability_score > 28.0937`，页面显示为 `> 28.1` | 基于当前总体记录分布校准 |
| 信号质量 | `signal_quality_score > 33.5206`，页面显示为 `> 33.5` | 基于当前总体记录分布校准 |
| 稳定片段比例 | `stable_segment_ratio > 0.50` | 明确采用 50% 的业务阈值，不再强行校准至 25% 通过率 |

以上前三项是当前样本和当前预览算法下的展示阈值，不应被解释为外部设备、完整原始波形或临床应用的通用标准。算法口径或样本总体变化后，应生成新版本阈值并记录校准数据集版本。

每个寸、关、尺通道分别计算四项指标。一个通道只有四项均通过时，才标记为通道质量合格。

单条记录的局部通道质量标签如下：

| 标签 | 定义 |
| --- | --- |
| `质量合格` | 寸、关、尺三通道均满足四项质量规则，且记录级指标满足规则 |
| `1通道质量不足` | 仅一个通道未通过四项联合规则 |
| `2通道质量不足` | 两个通道未通过四项联合规则 |
| `三通道质量不足` | 三个通道均未通过四项联合规则 |

记录列表必须显示质量不足原因，包括失败维度以及对应通道，例如 `Cun[SNR/质量]`。个人 summary 必须统计局部通道未对准比例，即存在 1 或 2 个质量不足通道的记录占比。

## 3. 更新后的总体技术路线

前四阶段解决的是单条记录是否可靠、模板是否稳定、片段是否可用的问题。后续阶段要解决的是：同一患者不同时间的脉波模式是否变化，变化是否超过个人正常波动范围，早中晚是否存在稳定的状态偏移趋势，不同患者是否存在稳定的脉波表型差异，以及这些差异是否来自真实个体差异，还是来自设备适配、腕围差异和通道未对齐。

更新后的主线为：

周期性脉搏信号确认 → 去趋势波动性分析 → 寸关尺三通道模板提取 → 同次记录内模式稳定性分析 → 压力/幅值干扰剥离与特征标准化 → 患者级质量统计与设备适配分析 → 个人基线模板建立 → 早中晚漂移趋势分析 → 长期纵向趋势分析 → 不同患者差异分析 → 多模态状态关联分析 → 中医解释与医师复核层。

后续分析不再直接使用原始波形或原始幅值作为核心生理特征，而应基于以下特征体系：

| 特征类别 | 主要用途 |
| --- | --- |
| 压力/接触干扰特征 | 识别基础压力、角度、接触深浅、位移伪影和幅值缩放 |
| 归一化模板形态特征 | 描述去基线、去幅值后的脉波形态差异 |
| 模板拟合质量与残差波动特征 | 判断模板是否解释当前脉波，以及剩余非周期波动结构 |
| 横向相位偏移变异特征 | 描述 HRV/PRV-like 的模板相位变异和节律波动 |
| 通道间相位差与三通道结构特征 | 描述寸、关、尺之间的相对时序、形态和结构关系 |
| 设备适配与通道未对齐特征 | 识别固定尺寸传感器与患者腕围、腕部形态不匹配造成的伪差异 |

整体上形成三层分析框架。第一层是采集质量与设备适配层，用于判断数据能不能用、哪些通道可信、哪些患者存在设备适配风险。第二层是个体纵向状态层，用于分析同一患者早中晚变化、长期趋势和状态偏移。第三层是患者间差异层，用于分析不同患者之间的稳定脉波表型差异，并排除设备、腕围、通道未对齐等混杂因素。

## 4. 核心概念与指标体系

### 4.1 周期化脉搏信噪比

周期化脉搏信噪比用于判断某个通道是否真正采集到了脉搏周期性信号。其核心思想是将原始波形分解为低频趋势、周期性脉搏模板和非周期残差，然后计算周期性模板能够解释多少信号能量。

建议定义：

`periodic_snr = 周期性模板解释方差 / 非周期残差方差`

如果一个通道的 `periodic_snr` 高，说明该通道存在清晰的心搏周期性结构。如果一个通道波形非常平稳但 `periodic_snr` 很低，则提示该通道可能没有对准寸、关、尺位置，或者接触压力不足。

### 4.2 压力/幅值干扰剥离模型

由于传感器压在手腕上的角度不同，各通道基础压力不同，波形会上下偏移，波幅也不同。后续特征提取应将这些采集干扰与相对稳定的波形形态分开。

建议对每个通道、每个搏动或窗口拟合：

`y_c,n(φ) = b_c,n + a_c,n × T_c(φ - δ_c,n) + r_c,n(φ)`

其中，`c` 表示寸、关、尺通道，`n` 表示第 n 个搏动，`φ` 是归一化相位，`T_c` 是该通道模板，`b_c,n` 是垂直基线偏移，`a_c,n` 是幅值缩放，`δ_c,n` 是横向相位偏移，`r_c,n` 是残差。

`b` 和 `a` 主要反映基础压力、接触角度、压迫深浅和局部耦合状态。它们应作为压力/接触干扰特征记录，而不应直接进入生理或中医解释层。形态学特征应优先在去基线、去幅值后的标准化波形上提取：

`z_c,n(φ) = [y_c,n(φ) - b_c,n] / a_c,n`

### 4.3 去趋势波动性

去趋势波动性用于分析在去除低频趋势和周期性脉搏模板之后，信号中剩余的非周期波动。它可以反映采集噪声、体动伪影、姿态变化、呼吸调制、自主神经相关波动或身体状态变化。

建议同时计算残差波动强度和多尺度去趋势波动特征。残差波动强度包括 `residual_std`、`residual_cv`、`residual_energy_ratio`、`residual_entropy`。多尺度特征包括 `dfa_alpha`、`multi_scale_fluctuation_slope`、`windowed_residual_fluctuation`。DFA 或多尺度波动分析不应直接被解释为中医证候，而应作为“去趋势后非周期波动结构”的数学表征。

### 4.4 模板相位变异

在已经使用 SNR 最强通道统一三通道心搏相位的基础上，每个通道相对模板的横向偏移 `δ_c,n` 可以作为单次测量内的相位变异序列。它不应简单等同于 HRV，但可以作为 HRV/PRV-like 的脉波节律指标，用于分析同一次测量中脉波相位提前、滞后和波动模式。

关键判断是：横向偏移是否是生理信号，必须看它是否与压力伪影解耦。如果 `δ` 变化伴随 `baseline_jump`、`amplitude_scale_jump`、三通道同步突变和 `template_fit_r2` 下降，则更可能是位移伪影。如果 `δ` 变化连续、周期性存在、模板拟合良好，并且 `b/a` 稳定，则更可能是生理性节律调制。

### 4.5 寸关尺三通道个人脉搏模板

每次记录应为寸、关、尺三个通道分别提取平均搏动模板。模板分为幅度保留模板和幅度归一化模板。幅度保留模板用于观察寸、关、尺三部脉势强弱、相对能量分布和通道平衡关系；幅度归一化模板用于观察主波上升速度、峰宽、下降速度、重搏波位置、波形尖缓程度等形态特征。

长期高质量记录可用于建立每个用户的个人基线模板。后续每次记录都与该用户自己的基线模板比较，而不是优先与人群平均模板比较。

### 4.6 疑似未对准评分

由于最稳定的低幅信号可能只是未采到有效脉搏，因此需要建立 `alignment_suspicion_score`。该指标综合周期化脉搏信噪比低、脉搏周期性能量低、模板解释方差低、主波幅度过小、信号过度平稳、与该用户历史同通道模板差异过大等因素。如果某个通道同时表现为低能量、低 `periodic_snr` 和高平稳性，则不能判为高质量稳定信号，而应标记为疑似未对准或接触不足。

## 6. 分阶段开发计划与分析目标

### 阶段一：三通道数据结构与基础预处理（已完成）

阶段目标是建立以同一患者为主体、以寸关尺三通道为核心的数据结构，确保每次记录都能被拆解为 `measurement`、`channel`、`window`、`template` 和纵向状态等可追踪对象。

已完成能力包括原始脉诊波形导入、三通道数据结构、用户/设备/时间/通道索引、基础滤波、重采样、异常值标记和数据完整性报告。

验收状态：已完成。

### 阶段二：脉搏周期识别与周期化信号确认（已完成）

阶段目标是判断每个通道是否真正采集到周期性脉搏信号，并避免把低幅稳定信号误判为高质量信号。

已完成能力包括搏动周期检测、主波峰值识别、搏动相位归一化、周期模板初步提取、`periodic_snr`、`pulse_energy` 和 `alignment_suspicion_score` 计算。

验收状态：已完成。

### 阶段三：去趋势波动性与周期模板分解（已完成）

阶段目标是将每个通道的原始信号分解为低频趋势、周期性脉搏模板和非周期残差，并分析去趋势后的残差波动结构。

已完成能力包括低频趋势估计、搏动相位对齐、幅度保留模板、幅度归一化模板、残差序列计算、DFA 或多尺度波动分析，以及三通道模板图输出。

验收状态：已完成。

### 阶段四：同次记录内三通道波形模式稳定性分析（已完成）

阶段目标是分析同一次记录中寸、关、尺三个通道的脉波模式是否在时间上保持稳定，定位最能代表该次记录的真实脉波状态的有效片段，并识别模板偏移伪影和疑似位移事件。

已完成能力包括滑动窗口切分、窗口级周期模板提取、窗口级 `periodic_snr`、`template_coherence`、`residual_fluctuation`、通道特异性漂移检测、三通道同步变点检测、推荐有效片段、绿黄红偏移热度、事件标注和单条记录模板稳定性页面。

验收状态：已完成。

### 阶段五：特征提取体系标准化与特征表建设

阶段目标是在已完成单条记录模板提取的基础上，建立统一的脉诊特征表，为后续患者级质量统计、早中晚分析、纵向趋势、患者间差异和多模态关联提供标准输入。

分析目标包括剥离基础压力、角度、幅值缩放等采集干扰；提取与垂直偏移和绝对波幅相对无关的归一化波形特征；提取横向模板偏移和相位变异特征；提取残差波动和 DFA 类非线性特征；提取寸、关、尺三通道之间的结构关系。

核心任务如下：

| 任务 | 说明 |
| --- | --- |
| 标准化拟合模型 | 对每个通道、每个搏动或窗口估计 `b/a/δ/residual` |
| 压力干扰特征提取 | 提取 `baseline_offset`、`amplitude_scale`、`pressure_artifact_score` |
| 形态不变特征提取 | 在去基线、去幅值后的标准化波形上提取形态特征 |
| 相位变异特征提取 | 对 `δ` 序列计算均值、标准差、RMSSD、频域和 DFA 类指标 |
| 残差波动特征提取 | 对残差计算能量、熵、多尺度波动和 DFA 指标 |
| 三通道结构特征提取 | 计算寸关尺形态距离、相位差、相对能量和结构指数 |
| 特征表建设 | 建立记录级、通道级、搏动级、相位级、残差级特征表 |
| 特征质量审计 | 输出缺失率、异常值率、质量依赖性、设备敏感性 |

核心特征类别包括压力/接触干扰特征、归一化模板形态特征、模板拟合质量特征、残差波动特征、模板相位偏移特征、通道间相位差特征、三通道结构特征和记录级综合特征。

主要交付物包括 `record_feature`、`channel_feature`、`beat_level_feature`、`phase_variability_feature`、`residual_fluctuation_feature`、`channel_structure_feature`、特征计算 API 和特征质量检查报告。

验收标准：每条高质量记录均能生成标准化特征；每个通道均能区分压力/接触干扰特征与形态、相位、残差类特征；后续分析模块不再直接依赖原始幅值作为核心解释指标；特征表支持按用户、记录、通道、时间段和设备查询。

### 阶段六：患者级脉诊质量统计与设备适配分析

阶段目标是在数据概览页面中按患者统计脉诊质量，并识别由于固定传感器尺寸、腕围差异、手腕形态差异导致的长期通道未对齐问题。

分析目标包括统计每个患者的整体质量通过率，统计每个患者四个质量维度的通过率，识别患者级长期通道未对齐模式，分析腕围、手腕宽度、设备编号与通道质量之间的关系，计算患者设备适配风险评分。

核心指标包括：

| 指标 | 说明 |
| --- | --- |
| `quality_pass_rate` | 患者记录级质量通过率 |
| `snr_pass_rate` | 周期 SNR 通过率 |
| `template_stability_pass_rate` | 模板稳定性通过率 |
| `signal_quality_pass_rate` | 信号质量通过率 |
| `stable_segment_pass_rate` | 稳定片段比例通过率 |
| `cun_alignment_suspicion_rate` | 寸部疑似未对准比例 |
| `guan_alignment_suspicion_rate` | 关部疑似未对准比例 |
| `chi_alignment_suspicion_rate` | 尺部疑似未对准比例 |
| `patient_device_fit_risk_score` | 患者设备适配风险评分 |
| `persistent_channel_failure_pattern` | 长期通道未对齐模式 |

建议的设备适配风险评分为：

`patient_device_fit_risk_score = 0.35 × max_channel_alignment_suspicion_rate + 0.25 × low_snr_channel_ratio + 0.20 × low_energy_channel_ratio + 0.20 × channel_imbalance_persistence_score`

主要图表包括患者质量通过率排名图、患者 × 质量维度热力图、患者 × 通道未对齐热力图、SNR × 稳定片段比例散点图、腕围 × 周期 SNR 散点图、腕围 × 疑似未对准比例散点图、设备适配风险患者排名图和通道未对齐模式分布图。

主要交付物包括数据概览页脉诊质量分析模块、患者质量统计表、设备适配风险评分模型、通道未对齐模式统计报告和腕围相关分析入口。

验收标准：数据概览页能够显示每个患者寸、关、尺三通道的疑似未对准比例；系统能够识别某个患者是否长期存在固定通道未对齐；如有腕围数据，系统能够展示腕围与通道 SNR、脉搏能量、疑似未对准比例之间的相关关系；高设备适配风险患者被明显标记；长期疑似未对齐通道不默认纳入个人模板、早中晚变化和多模态相关主分析。

### 阶段七：个人脉波基线模板与正常波动范围建立

阶段目标是为每个患者建立自己的寸、关、尺三通道个人基线模板，并计算个人正常波动范围，使后续早中晚分析和长期趋势分析以个人为参照。

分析目标包括筛选每个患者的高质量记录，建立个人三通道归一化形态模板，建立个人相位偏移基线，建立个人残差波动基线，建立个人通道结构基线，计算每个特征的个人正常波动范围。

核心指标包括 `personal_baseline_template`、`personal_shape_feature_baseline`、`personal_phase_variability_baseline`、`personal_residual_fluctuation_baseline`、`personal_channel_structure_baseline`、`personal_normal_range` 和 `baseline_deviation_score`。

核心任务如下：

| 任务 | 说明 |
| --- | --- |
| 高质量记录筛选 | 排除低 SNR、低模板拟合、低稳定片段、疑似未对准和高压力伪影记录 |
| 通道级基线建立 | 分别建立寸、关、尺个人模板和特征中位数 |
| 形态基线建立 | 对归一化模板形态特征建立个人中心和正常范围 |
| 相位基线建立 | 对 `phase_shift` 序列统计建立个人节律波动范围 |
| 残差基线建立 | 对残差波动、熵、DFA 等建立个人正常波动范围 |
| 三通道结构基线 | 建立三通道形态距离、相位差和能量结构基线 |
| 偏离评分 | 每次新记录计算相对个人基线的偏离度和偏离来源 |

主要交付物包括个人脉波模板库、个人特征基线表、个人正常波动范围表和个人脉波指纹报告。

验收标准：每个有足够高质量记录的用户均可生成个人基线模板；每次新记录均可与个人基线比较；系统能够识别某次记录偏离个人基线的主要通道和主要原因；基线建立过程排除长期疑似未对齐通道和高压力伪影记录。

### 阶段八：同一患者早中晚漂移趋势分析

阶段目标是分析同一患者早、中、晚不同时间段的脉波状态变化，区分采集漂移、相位漂移、形态漂移和残差波动变化。

分析目标包括判断早中晚是否存在周期 SNR 差异，判断早中晚是否存在压力/接触伪影差异，判断早中晚是否存在模板相位偏移差异，判断早中晚是否存在归一化形态特征差异，判断早中晚是否存在残差波动差异，判断变化是否超过个人正常波动范围。

需要明确区分四类漂移：

| 漂移类型 | 主要特征 | 解释边界 |
| --- | --- | --- |
| 采集漂移 | `baseline_offset`、`amplitude_scale`、`pressure_artifact_score`、`alignment_suspicion_score` | 优先解释为压力、角度、佩戴或设备适配问题 |
| 相位漂移 | `phase_shift_mean`、`phase_shift_sd`、`phase_shift_rmssd`、`phase_shift_trend_slope` | 可能包含 PRV-like 或脉搏传导信息，但必须排除压力伪影 |
| 形态漂移 | `template_peak_phase`、`rise_time_ratio`、`half_width_ratio`、`notch_phase`、`template_distance_to_baseline` | 更适合作为脉波状态变化核心特征 |
| 残差波动漂移 | `residual_energy_ratio`、`residual_entropy`、`residual_dfa_alpha` | 反映模板之外的非周期波动或状态不稳定 |

核心指标包括 `time_of_day_phase_shift`、`time_of_day_shape_deviation`、`time_of_day_residual_fluctuation`、`time_of_day_pressure_artifact_score`、`time_of_day_channel_balance_shift`、`morning_baseline_deviation`、`noon_baseline_deviation` 和 `evening_baseline_deviation`。

分析方法应优先采用个体内标准化：

`feature_z = (当前记录特征 - 该患者个人基线中位数) / 该患者个人基线 MAD`

如果数据量允许，可进一步采用控制设备、通道和质量协变量的模型：

`feature ~ time_of_day + device + channel + quality_covariates + (1 | user)`

主要图表包括早中晚三通道模板叠加图、早中晚 `phase_shift` 箱线图、早中晚归一化形态特征雷达图或条形图、早中晚残差波动趋势图、早中晚压力伪影对比图、早中晚相对个人基线偏离热力图。

主要交付物包括患者早中晚漂移趋势分析页、早中晚差异统计表和早中晚状态偏移报告。

验收标准：能够展示一个用户同一天或多天早中晚三次的三通道脉波差异；能够区分真实状态偏移和低质量采集造成的假差异；能够说明变化主要发生在寸、关、尺中的哪个通道；能够说明变化主要属于采集漂移、相位漂移、形态漂移还是残差波动漂移；早中晚差异结论必须标记是否超过个人正常波动范围。

### 阶段九：同一患者长期纵向趋势分析

阶段目标是基于个人基线和正常波动范围，分析同一患者在多天、多周或更长时间尺度上的脉波变化趋势。

分析目标包括识别患者长期形态偏离趋势，识别患者长期相位变异趋势，识别患者长期残差波动趋势，识别患者长期三通道结构变化，识别连续偏离个人基线的状态变化事件，区分真实纵向趋势和单次低质量采集。

核心指标包括 `longitudinal_shape_deviation`、`longitudinal_phase_variability_trend`、`longitudinal_residual_fluctuation_trend`、`longitudinal_channel_structure_trend`、`rolling_baseline_deviation`、`state_change_event_score` 和 `state_change_event_duration`。

核心任务如下：

| 任务 | 说明 |
| --- | --- |
| 用户级纵向序列构建 | 按日期、早中晚、设备和通道组织记录 |
| rolling baseline | 用过去 N 次高质量记录形成滚动个人基线 |
| 偏离度计算 | 每次记录计算相对个人基线的形态、相位、残差和结构偏离 |
| 趋势估计 | 使用稳健回归或滑动窗口估计长期趋势 |
| 状态事件识别 | 识别连续超过个人正常范围的偏离区间 |
| 质量过滤 | 排除低质量、疑似未对准和高压力伪影造成的假趋势 |
| 时间段分层 | 区分早中晚时间段造成的日内结构差异 |

主要图表包括个人长期脉波趋势图、寸关尺三通道长期偏离曲线、相位变异长期趋势图、残差波动长期趋势图、通道结构长期趋势图和状态变化事件时间轴。

主要交付物包括个人长期趋势分析页、状态变化事件检测模块和长期趋势统计报告。

验收标准：系统能够展示一个人长期脉波状态是否稳定；能够识别某一时期是否出现持续偏离个人基线；能够区分单次低质量采集与真实纵向趋势变化；能够展示变化主要来自寸、关、尺哪个通道，以及来自形态、相位、残差还是三通道结构。

### 阶段十：不同患者之间的差异分析

阶段目标是分析不同患者之间是否存在稳定的脉波表型差异，并区分真实个体差异、状态波动差异和设备适配差异。

分析目标包括比较不同患者的归一化形态特征，比较不同患者的相位变异特征，比较不同患者的三通道结构特征，计算个体内距离与个体间距离，判断哪些特征具有患者区分能力，识别由腕围和通道未对齐造成的伪差异。

不同患者之间不建议直接比较原始幅值。原始幅值受压力、角度、接触面积和通道基础压力影响过大。优先比较归一化形态特征、相位变异特征、三通道结构特征、个人模板稳定性特征和设备适配风险特征。

核心指标包括 `within_subject_distance`、`between_subject_distance`、`between_within_ratio`、`subject_separability_score`、`personal_template_distance_matrix`、`shape_feature_patient_variability`、`phase_feature_patient_variability` 和 `channel_structure_patient_variability`。

核心判断为：

`between_within_ratio = between_subject_distance / within_subject_distance`

如果某类特征的 `between_subject_distance` 明显大于 `within_subject_distance`，说明该特征更适合作为患者脉波表型。如果某类特征的 `within_subject_distance` 很大，说明该特征更像状态波动指标，而不是稳定体质指标。

主要图表包括患者个人模板距离矩阵、个体内距离 vs 个体间距离分布图、患者形态特征 PCA 或 UMAP 图、患者相位变异特征分布图、患者设备适配风险对照图和伪差异排查图表。

主要交付物包括患者间差异分析页、患者脉波表型分析报告、可用于患者区分的稳定特征清单和需排除的伪差异特征清单。

验收标准：能够区分稳定个体差异、状态波动差异和设备适配差异；不同患者比较默认使用归一化形态、相位、残差和三通道结构特征，而不是原始幅值；长期疑似未对齐通道不参与患者间核心差异分析；能够输出适合患者区分的特征和不适合解释的伪差异特征。

### 阶段十一：多模态状态关联分析

阶段目标是将脉象纵向特征与舌象、面象、问诊、睡眠、运动、饮食等其他指标对齐，分析同一患者内部的多模态同步变化。

分析目标包括分析脉象形态偏离是否与舌象变化同步，分析相位变异是否与睡眠、运动、情绪等状态相关，分析残差波动是否与问诊症状波动相关，区分个体内相关和个体间相关，避免将人群差异误解释为状态相关。

核心指标包括 `within_subject_multimodal_correlation`、`between_subject_multimodal_correlation`、`repeated_measures_correlation`、`pulse_tongue_sync_index`、`pulse_face_sync_index`、`pulse_symptom_sync_index` 和 `multimodal_state_event_score`。

核心任务如下：

| 任务 | 说明 |
| --- | --- |
| 时间对齐 | 对齐脉象、舌象、面象、问诊、睡眠、运动和饮食记录 |
| 样本构建 | 构建同一患者多模态纵向样本表 |
| 个体内相关 | 对同一患者内部的脉象偏离和其他模态变化计算相关 |
| 个体间相关 | 计算长期个体差异相关，但与个体内相关分开报告 |
| 重复测量相关 | 使用 repeated-measures correlation 或个体中心化相关 |
| 多模态同步指数 | 判断多个模态是否在同一时间段发生状态偏移 |
| 状态事件视图 | 将脉象、舌象、面象和问诊变化放在同一时间轴上展示 |

主要图表包括脉象—舌象纵向同步图、脉象—面象纵向同步图、多模态状态时间轴、个体内相关与个体间相关对比图和多模态特征网络图。

主要交付物包括多模态纵向状态分析页、多模态关联统计表和多模态状态事件报告。

验收标准：能够展示同一患者内部多模态指标是否同步变化；不会把人群差异误解释为个体状态变化；能够将脉波状态偏移与其他模态变化放在同一时间轴上观察；多模态结论必须标记为探索性或验证性。

### 阶段十二：中医解释层与医师复核界面

阶段目标是将算法特征转化为中医师可理解的寸、关、尺三部脉波状态描述，并提供医师复核入口。

分析目标包括将三通道结构特征映射为寸关尺三部模式，将归一化形态特征映射为脉形变化描述，将相位变异特征映射为节律波动描述，将残差波动特征映射为脉波状态不稳描述，将长期偏离个人基线解释为状态变化线索。

输出原则是：不直接输出证候诊断，不将单一算法指标解释为疾病，所有解释均以“脉波状态变化”形式呈现，并提示需结合舌象、面象、问诊和医师判断。

核心任务如下：

| 任务 | 说明 |
| --- | --- |
| 解释映射表 | 建立算法指标到中医观察语言的映射关系 |
| 三部脉势展示 | 展示寸、关、尺三部的结构、形态和偏离信息 |
| 状态解释模板 | 输出非诊断性的脉波状态描述 |
| 医师复核 | 提供医师标注、修改、确认和反馈入口 |
| 数据回流 | 将医师反馈写入复核数据表，用于后续算法校准 |

主要交付物包括中医解释映射表、脉波状态解释报告、医师复核标注界面和医师反馈数据回流机制。

验收标准：解释语言可被中医师理解；解释结果不越界为自动诊断；能够明确提示需结合舌象、面象、问诊和医师判断；医师反馈能够回流到后续特征校准和解释规则优化。

## 7. 页面更新规划

### 7.1 数据概览页

数据概览页应继续保留现有质量面板，并新增设备适配与通道未对齐分析。新增内容包括患者 × 通道未对齐热力图、腕围 × 周期 SNR 散点图、腕围 × 疑似未对准比例散点图、设备适配风险患者排名图、通道未对齐模式分布图，以及缺失腕围数据提示。

如果暂时没有腕围数据，页面应显示：“当前缺少腕围数据，暂无法分析设备尺寸适配性。系统已统计各患者长期通道未对齐模式，可在补充腕围数据后进行相关分析。”

### 7.2 脉诊分析用户 summary 页

用户 summary 页应在当前记录数、平均脉率、平均脉力、纵向周期 SNR、模板稳定性、疑似未对准比例等基础上，新增个人脉波模板、个人特征基线、设备适配风险提示、早中晚趋势摘要和长期趋势摘要。

个人典型脉波模板图应同时显示幅度保留模板和幅度归一化模板。能量与偏移散点图应继续用于观察该患者各条记录的能量水平与偏移伪影之间的关系，但后续解释应加入压力/接触干扰提示。

### 7.3 患者详情页

患者详情页保留单条记录模板稳定性分析，并新增以下页签：

| 页签 | 主要内容 |
| --- | --- |
| 单条记录分析 | 三通道波形、模板叠加、偏移伪影、疑似位移事件 |
| 特征摘要 | 归一化形态、相位变异、残差波动、压力伪影特征 |
| 早中晚漂移趋势 | 早中晚采集漂移、相位漂移、形态漂移、残差漂移 |
| 长期趋势 | 个人基线、rolling baseline、长期状态偏离、状态事件 |
| 设备适配 | 通道未对齐、腕围相关、设备适配风险 |

患者间分析页用于人群级研究，不应放在单个患者详情页内部。建议新增患者模板距离矩阵、患者特征空间分布图、个体内/个体间距离对比图、患者设备适配风险对照图和伪差异排查图表。

## 8. 开发优先级

### P0

P0 阶段必须完成特征表标准化、患者质量统计、设备适配与通道未对齐统计、个人基线模板建立和早中晚漂移趋势分析。这些能力构成后续纵向分析和患者间差异分析的基础。

### P1

P1 阶段完成长期纵向趋势分析、患者间差异分析、个体内距离与个体间距离分析和状态变化事件检测。P1 之后，系统应能够从“单条记录分析”升级为“患者级纵向状态分析”和“患者间表型差异分析”。

### P2

P2 阶段完成多模态状态关联分析、中医解释层、医师复核界面和多模态状态事件报告。P2 结果应以探索性分析和医师复核为主，不应直接输出自动诊断结论。

## 9. 风险与控制策略

### 9.1 原始幅值误解释风险

原始幅值差异优先解释为采集压力、接触角度和通道基础压力差异。只有在压力/接触干扰被控制、通道质量合格且长期重复出现时，才可作为辅助解释特征。

### 9.2 横向相位偏移误解释风险

模板相位偏移可能包含 PRV-like 生理信息，也可能来自位移伪影。必须同时检查 `baseline_offset`、`amplitude_scale`、三通道同步突变和模板拟合质量。只有在压力/幅值稳定、模板拟合良好且相位变化连续时，才作为相位变异特征使用。

### 9.3 设备适配伪差异风险

固定尺寸传感器可能与部分患者腕围、腕部宽度和桡动脉位置不匹配，导致某个通道长期质量不足。长期通道未对齐不应解释为某一部脉象长期偏弱，而应优先解释为设备适配或佩戴位置问题。

### 9.4 早中晚采集条件混杂风险

早中晚差异可能来自佩戴习惯、设备、操作者、采集时间不严格、饮食运动状态或信号质量差异。早中晚分析必须控制设备、通道、质量协变量和压力/接触伪影，并优先采用个人基线对照。

### 9.5 多模态伪相关风险

脉象、舌象、面象之间的相关可能来自个体差异，而不是同一患者状态同步变化。多模态分析必须区分个体内相关和个体间相关，并优先展示同一患者纵向同步变化。

## 11. 总结

前四阶段已经完成了单条记录的三通道模板提取、周期相位统一、模板稳定性分析、偏移伪影检测和疑似位移事件展示。后续开发应从“记录级质量分析”转向“特征驱动的患者级纵向分析”。

核心路线是：先建立标准化特征表，再做患者级质量和设备适配分析，再建立个人基线，再分析早中晚漂移趋势和长期趋势，再分析不同患者之间的稳定差异，最后进行多模态关联和中医解释层建设。

整个后续阶段的关键原则是：原始幅值差异优先作为采集压力和接触差异处理；归一化形态特征用于分析脉波形态变化；横向模板偏移特征用于分析 HRV/PRV-like 的节律变异；残差波动特征用于分析非周期状态波动；三通道结构特征用于分析寸关尺个人脉波模式；设备适配和通道未对齐必须作为患者间差异分析的混杂因素单独处理。
